# Configuring the Matching Options Tab

1. Create a filter expression for **Criteria for Matching From**, which controls which records search for matches.
2. Create a filter expression for **Criteria for Matching To**, which controls which records are candidates to be searched.
3. If desired, click the checkbox for **Leave Unique Records Unmatched**.
4. Select one of the following for **Matching Priority**:
   * **Prioritize Order of Sets**— Prioritizes sets based on their order in the strategy.
   * **Prioritize Highest Score** — Prioritizes the highest total match score.
