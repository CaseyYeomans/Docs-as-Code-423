# Designing FastApps in Profisee Portal

**FastApps** allow administrators to create customized portal applications within the Profisee Portal. These applications provide tailored views and interactions with master data to meet specific business needs. FastApps can include:

* Customized menu systems
* Web-based entity grids
* Reports
* Task notifications

Profisee administrators can create and design FastApps in Profisee Portal. Using the **App Designer** tab in the **Administration**section of the portal, you can create, import, export, and manage FastApps to be used elsewhere in the portal.![](https://Profisee.magentrixcloud.com/sys/staticasset/read/file-i32fdb8523e439b56.png)

From here, a list of all created FastApps is displayed with the following information:

* **Name**: The name given to the FastApp
* **Updated by**: The last user who edited the entity
* **Updated on**: The datetime the entity was last edited
* **Created by**: The user who created this entity
* **Created on**: The datetime the entity was created

If desired, you can right-click a matching strategy (or select multiple) in this list to bring up the context menu, allowing you to **Export** or**Delete**the selected matching strategies. If only selecting a single strategy, you can also **Open** or **Clone.**![](https://Profisee.magentrixcloud.com/sys/staticasset/read/file-i5ab04dfa5f226da5.png)

You can also use the Quick Filter to search for specific items in the FastApp list.![](https://Profisee.magentrixcloud.com/sys/staticasset/read/file-ibd219a242148f24c.png)
