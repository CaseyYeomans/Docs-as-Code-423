# Task List Control

The Task List control displays workflow tasks for process management. You can select or deselect the checkbox next to each attribute to enable or disable the task.

![](https://Profisee.magentrixcloud.com/sys/staticasset/read/file-i55b0a80b0787da10.png)

Configuration options:

* **Name**: Administrative reference name (not visible to end users)
* **Column Name**: The column name, based on the attribute associated with it.
* **Display Name**: The name displayed to users
* **Display Width**: Configure the display width for each column
* **Row Order**: Set the preferred order of rows
