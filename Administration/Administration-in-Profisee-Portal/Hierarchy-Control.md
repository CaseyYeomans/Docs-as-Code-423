# Hierarchy Control

### Hierarchy Control

The Hierarchy control displays hierarchical entity relationships in a tree structure.

![](https://Profisee.magentrixcloud.com/sys/staticasset/read/file-i71641c35235c1378.png)Configuration options:

* **Name**: Administrative reference name (not visible to end users).
* **Hierarchy**: Select the specific hierarchy to display from the dropdown menu.
