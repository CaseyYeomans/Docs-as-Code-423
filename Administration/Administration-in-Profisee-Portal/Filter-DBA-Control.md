# Filter: DBA Control

### Filter: DBA Control

DBA Filter controls allow users to narrow down data displayed in linked Entity Grid controls or other filter controls.

![](https://Profisee.magentrixcloud.com/sys/staticasset/read/file-i3e8ea5e962aece3b.png)

#### Basic Configuration

* **Name**: Administrative reference name (not visible to end users)
* **Label**: User-friendly display name shown in the portal
* **Domain Entity**: Select the domain entity containing filter values
* **DBA Input Mode**: Select how users interact with the filter:
  + **Dropdown Input**: Typeahead search field
  + **Visual (Chart) Input**: Interactive chart-based filtering
  + **Text Input**: Search field allowing users to type values (searches both Name and Code)
* **Display Format**: Choose how domain values are displayed:
  + Code
  + Code [Name]
  + Name
  + Name [Code]
* **Presentation View**: Select a presentation view for the page display
