# Report Viewer Control

### Report Viewer Control

The Report Viewer control embeds external reports within your FastApp page.

![](https://Profisee.magentrixcloud.com/sys/staticasset/read/file-i2d983c298e1bc40a.png)

Configuration options:

* **Name**: Administrative reference name (not visible to end users)
* **URL**: Complete URL of the external report to be displayed within the portal page.
