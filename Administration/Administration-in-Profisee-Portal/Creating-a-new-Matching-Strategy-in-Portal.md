# Creating a new Matching Strategy in Portal

You can create new matching strategies from the **Strategy** tab of the Match section of Profisee Portal. **Matching strategies** specify matching criteria and create control attributes that implement and monitor the matching process.

To create a new matching strategy:

1. Click the **+** icon on the toolbar in the Strategies tab. The **Add New Matching Strategy** pane opens. ![](https://Profisee.magentrixcloud.com/sys/staticasset/read/file-iff62ef27516e2063.png)
2. Input a **Name** for the strategy and select an **Entity** from the dropdown for the strategy to run against. Click **Continue**.![](https://Profisee.magentrixcloud.com/sys/staticasset/read/file-i4b86fcb5935ce186.png)

From here the Strategy Configuration view opens, allowing you to configure your strategy using the tabs for each section at the top. Once you are finished making changes to the strategy, click **Save** or **Save and Close** to finalize the changes.

## Strategy Configuration

Once you have created a matching strategy, you must configure settings on the various matching strategy tabs before the strategy can be effectively run. Available configuration tabs include:

* [Strategy](https://support.profisee.com/wikis/profiseeplatform/configuring_the_strategy_tab)
* [Output Attributes](https://support.profisee.com/wikis/profiseeplatform/configuring_the_output_attributes_tab)
* [Matching Sets](https://support.profisee.com/wikis/profiseeplatform/configuring_the_matching_sets_tab)
* [Matching Options](https://support.profisee.com/wikis/profiseeplatform/configuring_the_matching_options_tab)
* [Survivorship Sets](https://support.profisee.com/wikis/profiseeplatform/configuring_the_survivorship_sets_tab)
* [Survivorship Options](https://support.profisee.com/wikis/profiseeplatform/configuring_the_survivorship_options_tab)
* [Stewardship](https://support.profisee.com/wikis/profiseeplatform/configuring_the_stewardship_tab)
* [Lookup](https://support.profisee.com/wikis/profiseeplatform/configuring_the_lookup_tab)
* [Merge](https://support.profisee.com/wikis/profiseeplatform/configuring_the_merge_tab)
