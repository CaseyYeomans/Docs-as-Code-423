# Matching Administration in Profisee Portal

Profisee users with Super Adminsitrator or Matching Administrator roles with entity admin permissions can perform Matching Administration functions in Profisee Portal. Using the **Strategies** tab in the **Match**section of the portal, you can create, edit, import and export, and manage matching strategies.

![](https://Profisee.magentrixcloud.com/sys/staticasset/read/file-ib6c102f60213f8f.png)

From here, a list of all created Matching Strategies is displayed with the following information:

* **Name**: The name given for the Strategy
* **Entity**: The entity the matching strategy applies to
* **Status**: Whether the strategy is Enabled or Disabled.
* **Match**: Indicates that Matching is enabled
* **Survive**: Indicates that Survivorship is enabled
* **Auto-Match**: Indicates that Auto-Match is enabled. This automatically identifies clusters for uncluttered records.
* **Auto-Survive**: Indicates that Matching is enabled. This automatically performs survivorship as data is created, updated, and deleted.
* **Created by**: The user who created this entity
* **Created on**: The datetime the entity was created
* **Updated by**: The last user who edited the entity
* **Updated on**: The datetime the entity was last edited

If desired, you can right-click a matching strategy (or select multiple) in this list to bring up the context menu, allowing you to **Export**, **Delete**, **Enable**,or **Disable**the selected matching strategies. If only selecting a single strategy, you can also **Open**, **Clone**, **Run**, or **Clear**.![](https://Profisee.magentrixcloud.com/sys/staticasset/read/file-i67487bf02a7ccfb0.png)

You can also use the Quick Filter to search for specific items in the matching strategy list. You can click the ellipsis icon to import an existing matching strategy, or export all strategies in the strategy list.

![](https://Profisee.magentrixcloud.com/sys/staticasset/read/file-idba77087c699089.png)

> [!NOTE]
> You can import matching strategies from 2025 R2-2025 R4 as .MatchingStrategy files, or from 2026 R1 and after as .json files.
