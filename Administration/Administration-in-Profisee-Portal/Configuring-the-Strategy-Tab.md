# Configuring the Strategy Tab

The Strategy tab displays the basic configuration information for the matching strategy. Configure the following options on the Strategy Matching tab:

* **Strategy Name**: The name of the strategy.
* **Entity**: The Entity the strategy matches on. This cannot be edited after the strategy is created.
* **Clear Criteria**: An expression that determines the criteria for when the strategy clears.
* **Match**: Creates match groups based on the settings defined on the **Matching** tab.
* **Auto-Match**: Automatically matches unmatched records that are new or changed. Only available when Match is enabled.
* **Survive**: Creates golden records for match groups and survives attribute values from representative match cluster records as defined by settings on the **Survivorship** tab.
* **Auto-Survive**: Automatically survives new or changed match groups. Only available when Auto-Match and Survive are both enabled.
