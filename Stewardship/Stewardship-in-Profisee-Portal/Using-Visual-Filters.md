# Using Visual Filters

Visual Filters give Profisee FastApp Portal users the ability to view a visual breakdown of the attributes of their data and to filter their data with a single click. Visual Filters are available for entity grids or other filter controls that have been configured to include a Visual Filter.

Visual Filters can be configured to appear as the following types of graphs:

* Vertical Bar
* Horizontal Bar
* Pie Graph
* Donut Graph
* Tree Map

These graphs appear alongside the data within the entity for which they are configured. Each element in a graph represents a single attribute type, allowing a user to make comparisons at a glance.

Clicking an element in a graph filters an entity grid to only display records that contain that filtered attribute. A user can Ctrl+Click to select multiple filters from the same attribute type ('or' filter), or multiple filters across different attribute types ('and' filter) to curate the data to those specific attributes.
