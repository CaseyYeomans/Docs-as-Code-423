# Before You Begin

To use the Profisee Platform® you need:

* The URL for your Profisee instance (contact your MDM administrator)

* A user [account](https://support.profisee.com/wikis/profiseeplatform/accounts_overview) on your Profisee instance

* A [model](https://support.profisee.com/wikis/profiseeplatform/modeling_entities_in_profisee_portal_overview) and entity (as configured by your MDM administrator)

* Access to [Profisee Portal](https://support.profisee.com/wikis/profiseeplatform/portal_overview)
* (Optional) An [installation of FastApp Studio](https://support.profisee.com/wikis/profiseeplatform/install_profisee_FastApp_studio) on your Windows computer for advanced configuration

### Installation

If you are responsible for installing the server software then please refer to the relevant guide below:

* [On-premise / IaaS Install Guide](https://support.profisee.com/wikis/profiseeplatform/deployment_architecture)
* [PaaS / AKS Deployment Guide](https://support.profisee.com/wikis/profiseeplatform/deploying_profisee_using_the_arm_template)
* [PaaS / AWS Deployment Guide](https://support.profisee.com/wikis/profiseeplatform/profisee_aws_deployment_overview)
* If you are a ProfiseeCloud SaaS customer then the software instance will automatically be provisioned and installed for you

If you require assistance with installation, Profisee’s [support team](https://support.profisee.com/Force/force__case/index) is standing by to help.
